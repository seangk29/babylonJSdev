Documentation Page!

For element 5, I decided to create two scenes that differ in themes and resemble different topics. My first scene resembled Minecraft with the wood and grass textures on the spheres, ground and cubes. I used the createCube function to create the cubes and changed the x position to change where each cube is. I did the same for the sphere with the createSphere function. For scene 2 I based my scene of the solar system in the game the Outer Wilds. I added textures to each sphere to differeniate them.
